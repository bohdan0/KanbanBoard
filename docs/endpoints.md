# API Endpoints

## HTML API

### Root

- `GET /` - React app

## JSON API

### Users

- `POST /api/users`

### Sessions

- `POST /api/sessions`
- `DELETE /api/sessions`

### Lists

- `GET /api/lists`
- `POST /api/lists`
- `PATCH /api/lists/:id`
- `DELETE /api/lists/:id`

### Tasks

- `POST /api/tasks`
- `GET /api/tasks/:id`
- `DELETE /api/tasks/:id`
- `PATCH /api/tasks/:id`
