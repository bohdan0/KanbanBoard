# KanbanBoard

## Features

- [ ] User authentication, multisessions
- [ ] Lists, creating at the end of all Lists
- [ ] Lists, deleting
- [ ] Tasks, creating at end of each List
- [ ] Tasks, editing (changing title)
- [ ] Tasks, resolving (checkbox)
- [ ] Tasks, moving between columns (drag and drop)
